# Unifaat 2026 — Desenvolvimento Web

## Aula 02 — Node.js: Tipos primitivos, escopo léxico e imutabilidade  
Exemplos práticos para rodar no terminal com `node`.

### Estrutura do projeto

    .
    ├─ 01-escopo.js
    ├─ 02-tipos.js
    ├─ 03-operadores.js
    ├─ 04-event-loop.js
    └─ atividade-aula02.js

Cada arquivo é independente e pode ser executado sozinho.

### Requisitos

- Node.js 24

Para conferir:

    node -v

### Como executar

Na pasta do projeto:

    node 01-escopo.js
    node 02-tipos.js
    node 03-operadores.js
    node 04-event-loop.js
    node atividade-aula02.js

### Atividade da Aula

O arquivo `atividade-aula02.js` consolida os conceitos de tipos primitivos, escopo, imutabilidade e assincronismo em um único exercício prático.

### Objetivo da aula

- Entender escopo léxico e como `{}` define blocos
- Ver diferenças entre `var`, `let` e `const`
- Reconhecer tipos primitivos e usar `typeof`
- Entender `bigint` e `symbol`
- Praticar operadores matemáticos, lógicos e de comparação
- Visualizar assincronismo no Node com `setTimeout` e a ideia de fila no event loop
